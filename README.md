# 📅 우리학교 학사일정 캘린더

학교 학사일정을 쉽고 빠르게 확인할 수 있는 웹앱입니다.

학교를 검색한 뒤 월별 학사일정을 달력 형태로 확인할 수 있으며, 방학·입학/졸업·시험·일반 학사일정을 골라서 볼 수 있습니다. 모바일에서도 편하게 사용할 수 있도록 제작했습니다.

## 🌐 바로가기

https://sen-vip.github.io/school-calendar/

## 주요 기능

### 🏫 학교 검색

* 학교명 검색
* 시도교육청별 검색
* 선택한 학교 자동 저장
* 새로고침 후에도 선택 학교 유지

### 📅 학사일정 달력

* 월별 학사일정 조회
* 달력 형태 일정 확인
* 날짜별 일정 상세 보기
* 오늘 날짜 강조 표시

### 🔍 일정 찾기

* 방학 필터
* 입학·졸업 등 의식성 일정 필터
* 시험·평가 필터
* 일반 학사일정 필터
* 일정 키워드 검색

### 🎓 학년별 보기

* 학년별 학사일정 필터
* 전학년 일정 구분 표시
* 학년 대상 일정 확인 가능

### 📤 공유 기능

* 학교 링크 복사
* 이번 달 링크 복사
* 선택 날짜 링크 복사
* 학년 필터 상태까지 포함한 화면 공유
* 공유 링크 접속 시 학교·월·날짜·학년 보기 자동 복원

### 📱 모바일 지원

* 반응형 화면
* 모바일 최적화 UI
* 터치 기반 일정 확인

## 데이터 안내

본 서비스는 학교 학사일정 정보를 기반으로 동작합니다.

학교별 실제 일정은 변경될 수 있으므로 최종 내용은 학교 공지사항을 함께 확인하시기 바랍니다.

## 현재 버전


### Beta v3.5.1
* 일반 학사일정/기타 일정 기본색을 앱 대표색인 보라색으로 재정렬
* 입학식·졸업식·개학식·종업식·수료식·방학식은 하늘색으로 분리
* 학교 바꾸기 클릭 시 기존 달력을 숨기지 않고 검색 패널을 상태바 아래에 펼치도록 수정
* 검색 패널의 floating card 느낌을 줄이기 위해 투명·큰 그림자·sticky 겹침 효과 제거
* 학교 선택 후 검색 패널은 자동으로 접고 달력 위치로 이동

### Beta v3.5
* 선택한 학교가 있으면 상단 안내와 학교 검색 영역을 접고 달력을 바로 보여주도록 레이아웃 정리
* 현재 선택 학교는 얇은 상태바로 줄여 달력 시작 위치를 위로 조정
* 월 이동/오늘 버튼을 달력 제목 한 줄 안에서 정돈
* 날짜 칸 테두리는 과하게 색을 바꾸지 않고 일정 칩 중심으로 색상 표시
* 요약·필터·주요 일정은 달력 아래로 이동해 실제 확인 흐름을 개선

### Beta v3.4.3
* 보라색은 방학·개학·입학·졸업 등 핵심 학사일정 중심으로 사용
* 재량휴업일·개교기념일은 코랄 계열로 분리해 시험 빨강과도 구분
* 동아리·진로·자치·체험·지역사회연계 등 일반 학교행사는 블루 계열로 강화
* 기타로 분류되는 일정도 회색으로 묻히지 않도록 블루 계열 기본값 적용
* 토요휴업일은 회색 전용으로 유지

### Beta v3.4.2
* GitHub Pages/브라우저 캐시 때문에 기존 app.js/style.css가 남아 보이는 문제를 막기 위해 파일 버전 쿼리 추가
* 달력 렌더링 시 일정 제목 기준으로 최종 색상 분류를 한 번 더 적용
* 토요휴업일/공휴일/대체공휴일 필터·요약·달력칩 색상 일관성 보강

### Beta v3.4.1

* 프록시가 내려주는 기존 일정 type보다 화면 표시명 재판별을 우선 적용
* 토요휴업일을 회색 칩으로 분리
* 공휴일을 앰버 계열로 표시해 시험 일정 빨강과 구분
* 대체공휴일을 연한 앰버/노랑 계열로 별도 표시
* 방학·개학, 시험·평가, 행사·체험 색상 체계 정리
* 일정 종류 필터에 토요휴업·공휴일·대체공휴일 추가

## 업데이트 기록

### 3-5-1 업데이트

* 일반 학사일정은 보라색, 입학식·졸업식 계열은 하늘색으로 컬러 체계 재정렬
* 학교검색 패널을 달력 위에 겹치는 방식이 아니라 문서 흐름 안에서 펼쳐지도록 조정
* 선택 학교 상태바의 sticky 겹침을 제거해 스크롤 시 어지러움을 줄임
* 토요휴업일 외에는 회색 기본값으로 묻히지 않도록 일반 일정 기본값을 보라색으로 처리

### 3-5 업데이트

* 달력 우선 레이아웃으로 상단 영역 축소
* 학교 검색은 선택 전 중심, 선택 후에는 숨김 처리
* 선택 학교/공유 버튼을 컴팩트 상태바로 정리
* 달력 날짜 박스는 기본 테두리 중심, 일정 종류는 칩 색상으로 구분
* 공지 안내 박스를 주요 일정 아래로 내려 화면 첫 진입 시 달력 집중도 개선

### 3-4-3 업데이트

* 재량휴업일을 보라색에서 코랄 계열로 분리
* 일반 학교행사와 기타 일정의 기본 표시색을 블루 계열로 정리
* 방학·개학·입학식·졸업식·종업식·수료식은 핵심 학사일정으로 보라색 유지
* 노동절, 개교기념일, 진로·검사·자치·지역사회연계 등 키워드 분류 보강

### 3-4-1 업데이트

* 토요휴업일·공휴일·대체공휴일 색상이 기존 분류값에 밀려 잘못 보이던 문제 수정
* 일정 제목/내용 기준 재판별을 우선 적용해 어린이날·석가탄신일·대체공휴일 색상 분리 보강

### 3-4 업데이트

* 토요휴업일은 회색으로 표시해 반복 휴업 정보를 뒤로 정리
* 공휴일은 앰버, 대체공휴일은 연한 앰버로 표시
* 기말고사 등 시험·평가 빨강과 공휴일 색상이 충돌하지 않도록 분리
* 달력 칩, 일정 목록 배지, 월 요약 배지에 같은 색상 규칙 적용

### 3-3 업데이트

* 공유 기능 고도화
* 학교 링크, 이번 달 링크, 선택 날짜 링크 복사 지원
* 학년 필터를 적용한 상태도 공유 링크에 반영
* 공유 영역을 과한 박스 없이 가볍게 정리
* 초등학교 카드의 좌우 여백과 배지 정렬 보정
